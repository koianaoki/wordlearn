# 日本人向け英単語学習アプリ（2000語）要件メモ

## 1. ゴール
- 日本人学習者向けに、**2000語を登録した英単語カードアプリ**を作る。
- 学習体験はシンプルにし、まずは以下の最小機能に絞る。

## 2. 最小機能（今回これだけでOK）
1. 英単語カードを1枚表示する
2. カードをタップすると詳細を表示する
   - 例文（英語）
   - 日本語訳
3. カードの切り替えはスワイプ（左右）で行う
4. 表示する単語は登録済み2000語からランダム選出する

## 3. 技術方針
- フロントエンド: **SolidJS**
- 公開: **GitHub Pages（GitHub IO）**
- 単語データ: リポジトリ内のJSONで保持（2000語を登録）
- 例文/発音: **Free Dictionary API** から取得（取得できない場合はローカル情報を表示）

## 4. データ方針

### 4.1 ローカル単語データ（2000語）
- ファイル例: `data/words-ja-en.json`
- 最低限持つ項目
  - `id`
  - `word`（英単語）
  - `ja`（日本語訳配列）

サンプル:
```json
[
  {
    "id": 1,
    "word": "ability",
    "ja": ["能力", "才能"]
  }
]
```

### 4.2 API利用
- エンドポイント例: `https://api.dictionaryapi.dev/api/v2/entries/en/<word>`
- 主に使う項目
  - 例文（definitions[].example）
  - 発音情報（phonetics）
- 備考
  - 単語によっては例文がないことがあるため、その場合は「例文なし」で表示

## 5. 画面イメージ（最小）
1. 単語カード画面（メイン1画面）
   - 中央に英単語
   - タップで裏面（例文＋日本語訳）を表示
   - 左右スワイプで次/前のランダム単語へ

## 6. 実装順
1. SolidJSプロジェクト作成
2. 2000語JSONデータを配置
3. 単語カードUI実装（タップ反転）
4. スワイプ操作実装
5. ランダム選出ロジック実装
6. Free Dictionary API連携（例文・発音）
7. GitHub Pages公開

## 7. 今回の確定事項
- 単語数は**500ではなく2000語を登録**する。
- 機能は「カード表示・タップで例文/訳表示・スワイプ遷移・ランダム選出」に絞る。
